# goodguy-ranklist

## 简介

由于项目 https://github.com/ConanYu/FriendRatingServer 的缺陷而开的一个项目。

目前可以做到的：

- 管理你的关注人（friend）的Online Judge账号

- 用自己的网去获取你的关注人（friend）的各个Online Judge分数

## 目前的问题

- 很难在别人的电脑中使用

- 界面丑爆了

- 没有排序功能
